2.2           Login

 

To login to QuatroShelter you will need a User ID. and a password which will be assigned to you by the System Administrator.

 

 

Figure 2: Login Page

 

 

 

 

 

 

 

Label

Button or Link

Action

A

User ID.

Enter the User ID assigned by the System Administrator.

B

Password

Enter the User’s Password.

C

Login

Click this button to login with the User ID and Password entered.

D

Reset

Clear the Login Id and Password fields.

 

 

Enter your USER ID and your password in the spaces provided and then click the Login button. QuatroShelter Home page and menu will appear.


 

If you enter the incorrect User ID or password, access to the system will be denied and the message below will appear. Click Reset to clear the User ID and Password or just re-enter the User ID and password and try again.

 

Incorrect message.

 

Tip: You may change your password by clicking on the “Change Password” button on the Login page.
