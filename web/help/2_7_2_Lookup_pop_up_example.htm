2.7.2     Lookup pop-up example

 

Lookup pop-ups will display a list of the available options for the field being worked on. The user can click on the code wanted to make the selection. Some look-up pop-ups may have a filter function included on the pop-up to help on the search for information.

 

Figure 6: Lookup sample - Languages

 

 

The following describes a typical Lookup pop-up and its functions.

Label

Option

Action

A

Description Filter

The “Description” filter option finds records that match the code of the record being searched on the description of the records.

The User must enter text then click on the Search button to begin the search.

B

Item list

Click on any item description on the list to choose the item and populate it back in the form.

C

Search button

Press the search button to search the lists with the filter entered.
