2.12.8  Admission

 

Once clients have been assessed by the shelter staff and have approved the admission of a client, the user will need to process an admission for the client and will need to fill in the Admission Form. Be aware that an admission can be done from two areas in the system. An admission can be done right from the Intake Form and it can also be done from the Program Queue. At the Program’s queue the queue entry can be turned into an admission or the entry can be rejected.

 

Admissions list:

 

The admissions list show all the admissions done for a client and their status. Also Action buttons are available for the user such as the View button or the Update buttons.

 

 

Admissions form.

 

The Admission Form needs to be filled and a room and bed must be assigned for a client to be admitted to a program. Once the Admission Form has been saved, the client will be asked to sign the Admission Form using the signature pad. If the client does not want to sign the form, the field “Reason for not signing” must be filled in by the user. If a client does sign the Admission Form using the signature pad, the signature will be displayed on the screen and the user will be able to save it and later preview it if necessary.
