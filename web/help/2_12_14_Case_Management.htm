2.12.14                    Case Management

 

Case Management notes can be created and managed using QuatroShelter. Case notes usually require special security and therefore QuatroShelter allows case notes to be secured properly so that only the proper staff can add and changes notes or perhaps give some staff read only access to the notes.

 

The list of case notes can be viewed in a summary or detailed view. In summary view the page shows a one line per case note while the detailed view shows the whole note and its history.  

 

New notes can be started by pressing the New Note button. New notes can be assigned one or more Component of Service codes. The Component of Service codes will help group the notes is a systematic manner. Below is the current existing list of Components of Service codes.

 

 

A filtering function is also available for the search of case notes. The user can choose to select notes by data range, staff, case status or component of service.
