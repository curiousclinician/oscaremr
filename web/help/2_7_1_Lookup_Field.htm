2.7.1     Lookup Field

 

The use of Lookup Tables is often necessary as it ensures that the data entered in the system is categorized (or codified) in some manner. As opposed to a free text field, Lookup Tables contain pre-set options which force the user to choose from a fixed number of options.

       
       
 


 

Figure 5: Lookup Field

 

The following describes a typical lookup field.

 

Label

Option

Action

A

Select … text box

A text box is the area (or field) where the user is expected to fill the information being requested. In most lookup cases this text box is automatically field by the lookup selection.

B

Find

Click the Find Icon to pop-up the Lookup Table corresponding to the filed being filled.
