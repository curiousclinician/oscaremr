2.8.2     Drop-down list example

 

Drop-down lists will display a list of the available options for the field being worked on. The user can click on the code wanted to make the selection.

 

Figure 8: Drop-down list sample – Referred to

 

The following describes a typical drop-down list its functions.

Label

Option

Action

A

Field name

The drop-down list will normally have a field name such as “Referred to” in the example above.

B

Drop-down list field

The drop-down field with the drop-down button (down arrow). Press button to view available options on the list.

C

Drop-down list

The list of available options will be displayed as shown above. Click on item from the list to select the item wanted.
