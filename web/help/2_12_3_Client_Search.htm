2.12.3  Client Search

 

The process that should start whenever a client visits a shelter is that the client should be searched on the shelter system using the client search function. The client search function will help the user search the client across all the shelters in the system. The user can enter as much filtering information as he/she has. QuatroShelter will then search the system and display a list of all of the matching clients found.

If a client is found, QuatroShelter will determine how much information the user has access to based on the security established. If the client exists in the system, but the user has no access to the client’s information, the user will only get to see what has been establish as the client’s ‘basic information’. The client’s basic information includes: Last name, First name, Date of Birth, Gender, Aliases and Health & Safety message if applicable.
