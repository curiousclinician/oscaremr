1.1           Welcome

 

Welcome to QuatroShelter from Quatro Group.

 

QuatroShelter from Quatro Group is a shelter management system designed to assist shelter staff to manage and provide clients with quality services while enhancing the administration of the shelter system.

 

QuatroShelter is a web-based application with a single secured database.

 

QuatroShelter’s comprehensive access and data security model enables the application to allow secured access to the application, the functions in the application and the client information in the system.
