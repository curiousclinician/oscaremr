1         OVERVIEW

 

1.1   Welcome to QuatroShelters

 

QuatroShelter is a Shelter Management System designed to assist shelter staff with the day to day control and processing of shelter administration.  This system enables staff to provide quality client servicing, as well as enhanced administration in a secured environment.

 

QuatroShelter is a web-based application with a single secured database.

 

QuatroShelter’s comprehensive access and data security model enable the secured access of the application, the functions as well as the client’s information.

 

1.2   Modules Summary Table

 

QuatroShelter is divided into six main modules:

 

·         Client Management

·         Program Management

·         Facility Management

·         Reports

·         Administration

·         My Tasks

Client-Management

 

The Client Management Module provides the tools necessary to manage a client’s information.

Functions in this module include: Client search options, viewing transaction history of clients, intakes, admissions, referrals, discharges, service restrictions, complaints, consents, case management, as well as document attachment function and printing of labels.  Client Management can be accessed from the home page or through the Nav Menu by selecting Client.

 

 

 

 

Program Management

 

The Program Management Module provides the tools necessary to create and maintain service and bed programs offered through the shelters within their facilities.  The functions that can be performed within the Program Management Module include:  The program queue that handles incoming clients, function to viewing the clients’ currently active in each program, managing of incidents reported in a program, list of staff assigned to programs and managing of service restrictions in a program.  Program Management can be accessed through the Nav Menu or by selecting program management on the home page.

 

 

Facility Management

 

The Facility Management Module provides the tools necessary to create and maintain facilities within shelters and for the sector they serve.  The rooms and beds in facilities and programs are maintained using the facility function Room/Bed, capacity in the system is established by the definition of rooms and beds. Internal facility messages can also be managed through this module.  Facility Management can be accessed through the Nav Menu or by selecting facility management on the home page.

 

 

 

 

Reports

 

QuatroShelter has many reports that are available to users. The Report Module gives users access to these reports. The reports can be printed in several forms including PDF and Crystal formats.  Some of the main reports are:  Client Profile report, client servicing, service restrictions, incident summary, occupancy and facility and system messages, as well as user reports.  Report information can be viewed on the page, printed and or exported to a file for import into spreadsheets.

Features such as saving report templates and report options and criteria chosen make the reporting process simple.  Reports can be accessed through the Nav Menu or by selecting reports on the home page and can be secured for the staff as needed.

 

 

 

 

Administration

 

The Administration Module provides the tools necessary to create maintain and configure base areas of the system.  The areas maintained through administration include: User and role management, lookup tables maintenance, the SMIS ORG chart, merging of clients, system and facility messages maintenance and unlocking of users lock by the system.  Administration can be accessed through the Nav Menu or by selecting system administration on the home page.

 

 

 

 

My Tasks

 

Client tasks assigned to a user will show on the list of My Tasks for that user. The My Tasks function allows users to manage their tasks. Tasks that are assigned to a specific user can only be viewed by that user.

The My Tasks Module is accessible from almost any page in the system when accessing it through the Nav Menu.  It can also be accessed from the home page.
