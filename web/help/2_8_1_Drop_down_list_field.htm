2.8.1     Drop-down list field

 

The use of Drop-down Lists is often necessary as it ensures that the data entered in the system is categorized (or codified) in some manner. As opposed to a free text field, Drop-down Lists contain pre-set options which force the user to choose from a fixed number of options.
