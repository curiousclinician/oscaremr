8         Adding a New Program into the QUATROSHELTER System

 

a)     Click on the grey “Program” Tab:

 

Click “Program” Tab
 

 

 

b)      Click “Add Program”

 

Click “Add Program”

 

c)     You only need to fill in the  3 following fields:

a)     Name( of Program)

b)     Type (Bed or Service)

c)     Number of Max Participants that will be in the program

Then click “Save”

Fill in:

a)     Name

b)     Type

c)     Number of Max Participants in the program

 

Then click “Save”
 

 

 

 

d) A message will appear to show that the program has now been saved.
