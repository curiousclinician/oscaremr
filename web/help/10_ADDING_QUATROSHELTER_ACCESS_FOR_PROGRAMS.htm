10    ADDING QUATROSHELTER ACCESS FOR PROGRAMS

 

Each program in QUATROSHELTER must have the levels of access defined. Every agency has a number of roles, i.e. Housing worker, counselor, nurse. Each program must define what information, within that program, each role can have access to.

Ex: Agency A has staff that consists of counselors, nurses and intake workers. Program B within that agency must define the access for those roles. Program B will define access such that: nurses have access to the medical file, prescriptions (read-only) and all counselor and intake notes. Counselors will have access to counselor issues/notes, read-only nurse issues/notes and intake worker notes. Intake workers will have access to read-only counselor issues and intake issues/notes.
