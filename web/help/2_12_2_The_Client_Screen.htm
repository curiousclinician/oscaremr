2.12.2  The Client Screen

 

The client screen has four main areas. The navigator area, the heading area, the client identification information area and client information area.

 

 

 

Label

Option

Action

A

Navigator

The Navigator area has all of the available functions that can be accessed.

B

Heading

Client Management heading and the function selected.

C

Client Identification

The Client Identification area has some basic information about the client accessed. Enter the hour and minutes that the System Message should end.

D

Information area

The information area has the client’s information related to the function selected.
