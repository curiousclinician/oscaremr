2.11.6  Users for QuatroShelter

 

The users of QuatroShelter are created and maintained by the system administrator. There are two areas that have to be maintained. The first area is the user’s basic profile which includes the user’s User ID, names, password, phone, etc. The second area is the security setup for the user. That is what ORG the user has access to and what role the role performs within the ORG. See charts below to review functions of the system that pages involved.
