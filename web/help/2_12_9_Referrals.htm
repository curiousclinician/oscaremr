2.12.9  Referrals

 

QuatroShelter has a client referral function that allows users to create and maintain referrals for their clients.

 

Referrals can be created in several ways. A user can manually create a referral for client where the user would refer the client to some other program or agency. Some referrals are created automatically by QuatroShelter, for example when an intake is processed for a program the system will automatically do the referral to the program on the intake.

 

The referral list:

 

A history of all referrals for the client will be displayed when the user accesses the referrals function. New referrals may be started by pressing the New Referral button also the existing referrals on the list show the status of a referral and Action buttons are available to View or Update the referrals.

 

 

Referral page:

 

Referrals can be created and referrals can be done for any program on the system. Enter the information and press Save to save and process the referral.
