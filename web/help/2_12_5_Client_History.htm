2.12.5  Client History

 

The client history function shows a list of the client’s transactions that are being stored by the system. There are several transactions types being stored for this display by QuatroShelter, they are: intakes, admissions, referrals and discharges. The client search function is design to search a client across the whole shelters system. When a client is found, the system determines based on the security setup how much of the client’s information the user can access.
