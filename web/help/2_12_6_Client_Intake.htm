2.12.6  Client Intake

 

Every time a client arrives at a shelter requesting some type of service, the shelter staff must start the service process by processing an Intake. The Intake process includes the filling of the Intake Form.

 

 

Intake list.

 

When a user accesses the Intake function from the Navigator, the Intake list will display. The Intake list shows all of the Intakes that have been done for the client and their status. The user can start a new Intake or the Action buttons such as such as Update and View can be used.

 

 

 

 

Client Intake Form.

 

The Intake Form needs to be filled by the user when processing an intake. There is a few mandatory fields that must be filled in before the intake can be saved, these fields are: First name, Last name, Gender, Date of Birth and Program. User must fill the form and them press the Save button to save the Intake.

The following are will be updates when the intake is saved. The client history will be updated, an automatic referral will be processed and the queue of the program named on the Intake Form will be updated.

 

 

To save the Intake, just press the Save button on the Intake Form.

 

Once an intake is saved, the user will have some new options on the Intake Form. The user can start intake of the family members (for family shelters) or the user can proceed to admit the client by pressing the Admission button.
