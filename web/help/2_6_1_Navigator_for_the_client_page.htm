2.6.1     Navigator for the client page.

 

When in the client page, the Navigator lists the various pages (screens) associated with an individual client including the summary, history, intake, admission, referral, discharge, service restriction, complaint, consent, case management, attachment, tasks and print label pages. Some pages may not be accessible to all users based on the user’s security access level. See chart for further information.

Snow, James (206933)

Display of client’s first name, last name and client number.

B

Summary

By selecting this option the user will access the client’s summary page. The client’s summary page display the client’s basic information, the family members (if any), the active room and bed assigned to the client, the program(s) that the clients is active on and any active referrals for client.

C

History

By selecting this option the user will access the client’s transaction history page. The history page include a list of selected transactional information about the client including: Intakes, Referrals, Admissions and Discharges

D

Intake

By selecting this option the user will access the client’s intake pages. The intake pages include a list of all of the client’s intakes; the user can view the status of the intakes, view the detail of existing intakes, and create new intakes and process admissions from an intake.

E

Admission

By selecting this option the user will access the client’s admission pages. The admission pages include a list of all of the client’s admissions, the user can view the status of the admissions, view the detail of existing admissions, process new admissions which include the assignment of a room and bed and processing the client’s signature using the signature pad.

F

Referral

By selecting this option the user will access the client’s referral pages. The referral pages include a list of all of the client’s referrals; the user can view the status of the referrals, view the detail of existing admissions and process new referrals.

G

Discharge

By selecting this option the user will access the client’s discharge pages. The discharge pages include a list of all of the client’s discharges, the user can view the status of the discharges, view the detail of existing discharges, process new discharges.

H

Service Restriction

By selecting this option the user will access the client’s service restrictions pages. The service restrictions pages include a list of all of the client’s service restrictions, the user can view the status of the service restrictions, view the detail of existing service restrictions and process new service restrictions.

I

Complaint

By selecting this option the user will access the client’s complaint pages. The complaints pages include a list of all of the client’s complaints; the user can view the status of the complaints, view the detail of existing complaints and process new complaints.

J

Consent

By selecting this option the user will access the client’s consent pages. The consent pages include a list of all of the client’s consent forms, the user can view the status of the consent forms, view the detail of existing consent forms and process new consent forms which include the processing of the client’s signature using the signature pad.

K

Case Management

By selecting this option the user will access the client’s case management pages. The case management pages include a list of all of the client’s case management notes; the user can view the status of the case management notes, view the detail of existing case management notes and create new complaints.

L

Attachment

By selecting this option the user will access the client’s attachment pages. The attachment pages include a list of all of the client’s attachments already uploaded, the user view the detail of the existing attachments and attach new documents to the client file.

M

Tasks

By selecting this option the user will access the client’s tasks pages entered by the user. The tasks pages include a list of all of the client’s tasks; the user can view the status of the tasks, view the detail of existing tasks and process new tasks.

N

Print Label

By selecting this option the user will print a label with a bar code with the client’s number.
