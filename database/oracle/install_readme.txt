To setup the database, follow these step in order
	1. create user QUATROSHELTER 
	2. create user QUATROSHELTER_LOG
	
	both of the 2 users can be set in the built-in role of 
	
	login as QUATROSHELTER_LOG, 
	3. import from the file: 
		QuatroShelterDB_LOG.dmp  
	4. exexcute the SQL statements in the file:
		QuatroShelterDB_LOG.sql
	
	login as QUATROSHELTER, 
	5. import from the file: 
		QuatroShelterDB.dmp  
	6. exexcute the SQL statements in the file:
		QuatroShelterDB.sql
